## How to stuff 🤷‍

### Prerequisites

Node >10 (@todo add .nvmrc) 🛡

Yarn 😽



### Get running

clone this repo

```bash

$ git clone https://github.com/AlexanderCannon/node-fastify.git

```

Install with Yarn 🛠

```bash

$ yarn

```

Start the server ⚔️

```bash

$ yarn start

```
